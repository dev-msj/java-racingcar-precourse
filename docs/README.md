## Story
1. n대의 자동차 이름을 입력 받는다.
2. 몇 번 이동할 것인지 입력 받는다.
3. 각 자동차 별로 입력받은 횟수만큼 이동한다.
5. 각 회차별 이동 결과를 출력한다. 
6. 우승자를 출력한다.
