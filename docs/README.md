## Story
1. n대의 자동차 이름을 입력 받는다.
2. 몇 번 이동할 것인지 입력 받는다.
3. 0에서 9 사이의 random 값 추출한다.
4. 4이상은 전진, 3이하는 멈춘다.
5. 각 자동차 별로 3, 4번을 이동 횟수만큼 반복한다.
6. 각 회차별 이동 결과를 출력한다.
7. 우승자를 출력한다.
